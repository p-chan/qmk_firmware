# p-chan_pok3r

> A keymap for POK3R Layout 60% Keyboard.

![Keyboard](https://i.imgur.com/febEN2G.jpg)

## Features

- Function Keys
  - Leyer 1: `Caps Lock`
  - Leyer 2: `Caps Lock` + `menu`
- Arrow Keys
  - :arrow_up:: `Right Shift`
  - :arrow_down:: `menu`
  - :arrow_left:: `function`
  - :arrow_right:: `Right Control`
- Escape Key
  - `Caps Lock` + `esc`

## Other

- RGB Toggle: `CAPS LOCK` + `menu` + `Q`
- Reset: `CAPS LOCK` + `menu` + `esc`
